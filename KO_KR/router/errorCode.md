# MSDK백그라운드 에러 코드 #

## 1 개요 ##

1. MSDK 에러  코드는 연동층,로직층,플랫폼층으로 나뉩니다.
2. 일반 에러, 즉 appid 에러, appkey 에러, 권한 제어 등은 연동층에 의해 직접 복귀됩니다. 에러에 관한 자세한 내용은 msdk 연동층 에러 코드 부분을 참조 바랍니다.
3. 로직층에 에러 발생시 MSDK 연동층은 에러 코드 ret, 에러 설명 msg를 투명적으로 전송하며 여기에서 ret는 MSDK 로직층에 의해 정의됩니다.
4. 플랫폼층의 게임 에러와 연관될 경우 모바일QQtoken측 에러 MSDK는 ret：-10000에 복귀하고, msg는：플랫폼 에러 코드, 플랫폼 에러 설명으로 나타납니다. 예를 들어："msg":"-73,the user modify password wrong"
5. 플랫폼층의 게임 에러와 연관될 경우 위챗측 에러 MSDK는 ret：-20000에 복귀하고, msg는：플랫폼 에러 코드, 플랫폼 에러 설명으로 나타납니다. 예를 들어："msg":"40001,invalid credential"
6、MSDK연동층은 에러 코드에 대해 분류 처리를 완료한 상태이고 파라미터 err_type을 신규 추가하였습니다. 0:정상，1:네트워크가 불안정하므로 잠시 후 다시 시도해주세요. 2：권한을 다시 부여해 주세요. 3:refreshToken으로 acessToken을 리프레시해 주세요. 5:알 수 없는 에러가 발생하였습니다.
